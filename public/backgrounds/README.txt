هذا المجلد هو المصدر الرئيسي لجميع ملفات الخلفيات التي تُرفع من Background Manager.
لا يتم تخزين الملفات نفسها في قاعدة البيانات؛ يتم حفظ المسار فقط في جدول page_backgrounds.
This folder is the canonical source for all background assets uploaded via the Background Manager.
Only file paths/URLs are stored in the database (table: page_backgrounds), never the binary files themselves.
